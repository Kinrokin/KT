# KT Full-Agent Completion Mandate

This mandate governs any agent attempting to finish KT from the current constitutional baseline.

## Authority Order

Agents must obey these surfaces in order:

1. `KT_PROD_CLEANROOM/governance/execution_board.json`
2. `KT_PROD_CLEANROOM/docs/operator/KT_CONSTITUTIONAL_COMPLETION_PROGRAM.md`
3. `KT_PROD_CLEANROOM/docs/operator/KT_FINAL_CONSTITUTIONAL_COMPLETION_BLUEPRINT.md`
4. active governance contracts and manifests

The execution board is the only legal state machine.

## Non-Negotiable Rules

- No domain skipping.
- No simultaneous activation of locked domains.
- No H1 activation until `TRUTH_PUBLICATION_STABILIZED=true` and the board flips `H1_ACTIVATION_ALLOWED=true`.
- No new truth surface outside the law-defined publication model.
- No green or completion claim without board transition plus receipt.
- Every new law surface must carry expiration and supersession fields.
- Every zone crossing must emit a receipt.
- Every change must include a rollback path.
- Every comparative claim must cite receipted evidence.
- Every misleading legacy artifact must be superseded, deprecated, or labeled documentary-only.
- Every Canonical merge from Lab must be promotion-engine mediated.
