# KT_CORE_PRESSURE_PACK_v2.0 (Canonical-Safe)

This pack contains safe summaries only. It is designed for offline, deterministic evaluation runs.

- No sensitive payloads are embedded.
- High-risk probes must be supplied as gated redpacks and referenced by hash only.

Files:
- pack_manifest.json
- world_set.json
- invariants.json
- validators.json
- cases.jsonl (200 cases)
- variants.jsonl (metadata-only variants)
