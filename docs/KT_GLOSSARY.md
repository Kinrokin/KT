# KT Glossary

- **Kernel**: The sealed runtime under `KT_PROD_CLEANROOM/04_PROD_TEMPLE_V2/`.
- **Growth layer**: Tooling-only workflows under `KT_PROD_CLEANROOM/tools/growth/`.
- **Fail-closed**: Any ambiguity or violation halts; no silent substitution.
- **Append-only**: Evidence logs are written as JSONL and never overwritten.
