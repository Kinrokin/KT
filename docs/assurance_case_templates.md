# KT Assurance Case Templates

Every public, reviewer-facing, or operator-facing claim must use this structure:

Claim -> Argument -> Evidence -> Assumptions -> Limitations -> Tier

Evidence tiers:

- Tier 0: observed locally, unverified.
- Tier 1: receipted.
- Tier 2: replayed.
- Tier 3: external verifier reproduced.
- Tier 4: multi-run stable.
- Tier 5: commercially claimable.

G2, G3, G3.1, and G3.2 outputs are not Tier 5 unless external validation and commercial review explicitly authorize that status.
