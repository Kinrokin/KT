# KT Plain-Language Glossary

This glossary is for reviewer, buyer, and operator-facing explanations. It does not rename machine outcome IDs, receipts, validator constants, or canonical lane labels.

Current state in plain language:

```text
Independent external review attestation is pending. Internal launch-wedge, shadow, benchmark, runtime/context-efficiency, and adaptive-development work may continue under the allowed claims boundary.
```

| Internal machine term | External/plain-language term | Use |
| --- | --- | --- |
| Truth Lock | Current State Lock | Use for reviewer and buyer explanations of current canonical state. |
| Claim Ceiling | Allowed Claims Boundary | Use when describing which statements are supported by current evidence. |
| Forbidden Claims | Blocked Claims | Use for claims that current evidence does not support. |
| Superlane | End-to-End Workstream | Use for a complete multi-step delivery corridor. |
| Highway | Execution Lane | Use for a governed category of work with receipts and gates. |
| Gear | Internal Gate | Use for an intermediate step inside a workstream. |
| FP0 | Runtime/Context Efficiency Overlay | Use for no-claim-expansion runtime, local execution, and context-efficiency work. |
| Frontier Ingestion | Research Signal Intake | Use for non-authoritative research-pattern tracking. |
| Authority World Model | Claim Impact Simulator | Use for simulation of how a change could affect allowed claims. |
| Lobe | Specialist Module | Use for bounded specialist components. |
| Router | Routing Controller | Use for policy that selects a module or path. |
| Learned Router | Candidate Routing Policy | Use until evaluation proves a stronger routing policy. |
| Crucible | Evaluation Scenario | Use for preregistered stress or benchmark cases. |
| Epoch | Development Phase | Use for staged development or training periods. |
| Civilization Stack | Adaptive Orchestration Stack | Use for future adaptive modules; do not describe as production-complete. |
| S-tier | Category Leadership Claim | Blocked until readjudication evidence exists. |
| Beyond-SOTA | Do not use; blocked until independent benchmark evidence exists. | Blocked phrase for public, reviewer, and buyer materials. |
| 7B Amplification | Small-Model Substrate Ablation Study | Use for an ablation program only; proof remains unproven. |
| Commercial Activation | Commercial Claim Authorization | Use for claim authority, not payment or deployment availability. |
| External Reaudit | Independent External Review | Use while independent review acceptance remains pending. |
| External Attestation | Independent Review Attestation | Use for the outside reviewer artifact that KT must not self-author. |

Machine labels remain authoritative inside validators and receipts. Human-facing materials should use the external terms unless quoting a machine identifier.
