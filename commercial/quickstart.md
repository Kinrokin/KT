# KT Bounded Pilot Quickstart

1. Read `commercial/launch_boundary_notice.md`.
2. Inspect `external/evidence_bundle_manifest.json`.
3. Run the verifier or evidence-pack workflow selected for the pilot.
4. Compare every public sentence to `governance/allowed_launch_claims.json` and `governance/forbidden_launch_claims.json`.
5. Record feedback using `commercial/pilot_feedback_receipt.schema.json`.

The quickstart does not authorize commercial claims or external audit completion.
